# pressKeys
<span class="theme-doc-version-badge badge badge--secondary">production</span><br/><br/>

Press an array of keys one after another.

**Examples:** 
```typescript
await aui.pressKeys(['right', 'left', 'enter']);
```

   * @param \{PC_AND_MODIFIER_KEY[]} keys
